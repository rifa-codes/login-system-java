 Login System (Java)

 Project Description

This project is a simple **Login System** developed using Java.
It is a console-based application that allows users to log in using a username and password.

The system checks whether the entered credentials match the predefined values and displays a success or failure message.

 Features

*  User login authentication
*  Username and password validation
*  Displays error message for incorrect login
*  Beginner-friendly implementation
*  Simple console-based interface

 Technologies Used

* Java
* Core Java concepts
* Scanner class for user input

 Concepts Covered

* Classes and Objects
* Conditional Statements (`if-else`)
* String Comparison
* User Input Handling using `Scanner`
* Basic Authentication Logic

How to Run

1. Compile the program:

```java
javac LoginSystem.java
```

2. Run the program:

```java
java LoginSystem
```

 Sample Output

```java
===== Login System =====
Enter Username: admin
Enter Password: 1234

Login Successful
```
